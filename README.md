## 📖 About
This is a **satirical/experimental interface** that mimics an “unethical” AI assistant to teach:
- AI safety & alignment  
- Prompt injection risks  
- Content filtering failures  
- Responsible deployment practices  

It does **not** actually execute harmful actions, store user data, or bypass real-world safeguards.
