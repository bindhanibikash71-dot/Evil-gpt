## 🎯 WHAT IS THIS?
This repo is an **interface autopsy** of a rogue AI site. We break down exactly how it presents itself so that the security and dev community can recognize the red flags.

### 🧠 The Deceptive Pitch
The site's sole marketing line—**"No shackles. No rules."** —is a carefully crafted appeal. It directly targets users frustrated with the ethical safeguards on legitimate chatbots like ChatGPT or Claude. In the security world, this is a classic **social engineering dark pattern**.

---

## 🕵️ INTERFACE RED FLAGS (Educational)
Here’s what makes this interface immediately suspicious from a design perspective:

| 🔴 Element | ❗ Why It's Unethical |
| :--- | :--- |
| **"No Rules" Tagline** | Advertises the removal of safety filters, enabling harmful content. |
| **Dark, Aggressive Theme** | Aesthetic mimicry of "power without limits" to attract malicious users. |
| **Countdown Timer** | Creates false urgency (FOMO) to push users into quick, careless interaction. |
| **Minimalist UI** | No help, no privacy policy, no contact—a hallmark of throwaway malicious tools. |
